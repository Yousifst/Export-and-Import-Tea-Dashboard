This repository provides a Global Trade Analytics Dashboard designed for monitoring commodity import and export data. This interactive dashboard allows users to analyze market trends, track yearly performance, and identify key international trading partners.
Key Features
# Overall Performance Metrics: 
View total exported values, average/median unit prices, and overall year-over-year (YTY) changes.
# Trend Visualization:
Interactive line and bar charts display historical data from 2014 to 2026 to visualize trends in total value and YTY performance.
# Top Exporters Analysis:
A detailed table highlights leading exporting countries, their exported value, global market share, YTY change, and unit prices.
# Country-Specific Data:
Focus on specific countries like China and Sri Lanka to view detailed import/export trends.
Market Monitoring: Tools for selecting the number of countries to analyze and potentially identifying new markets.
Dashboard Data Snapshot
The dashboard presents data points such as:
Total Exported Value: $77.128 billion
Average Unit Price: $6.89K
Last Year Performance: $7.42 billion (-12.55% YTY change)
Top Exporter (China): $17.72 billion exported value, 22.98% global share
# Usage
This dashboard can be used by analysts, traders, and businesses to make informed decisions regarding logistics, pricing strategy, and market entry/expansion.
